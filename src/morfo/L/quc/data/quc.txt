Rox tanaj 

Chi utza'm le xukuq'ab' 

Jalajoj taq ub'ixik ucholajil le xukuq'ab' kaya' pa we uwachulew. Pa jun k'olib'al katzijoxik chi le xukuq'ab' ruk'a'm jun b'o'j chi utza'm nojinaq che q'ana pwaq. Jachike taq chomanik kab'ix pa le awokaj at chi rij le xukuq'ab'? Chatzijoj kuk' le awachi'l alab'om, alitomab' pa le tijob'al. 

Chujok chi utzukuxik jun xukuq'ab' 

La awilom uwach jun xukuq'ab'? Chatzijoj kuk' le awachi'l, jampa' awilom, xuquje' jawije' le awilom wi. 
Chawila' le wachib'al, xuquje' chatzijoj jas kawil che le xukuq'ab', le ub'anik, le unimal, chi'l le uka'yeb'al. 
Jas rajawaxik kib'an kuk' le awachi'l alitomab', alab'om, are chi kiwil jun xukuq'ab'? 

Le jab', xuquje' job' tzu

Pa jun q'ij, jun ala katajin kawa'kat pa taqle k'ache'laj, xaq k'ate' xumajij jun nimalaj jab'. Xutzukuj k'ut jawije' kapan wi, xa jeri' xuriq alaj taqjuyub' jacha ub'anik le xkanul. Are taq xub'an utzuka'yexik, xrilo k'o jujun alaj taq uchi' torotajinaq. Xok k'u b'i chi upam jun alaj juyub'.

Are taq xok b'ik, xeril e jupuq alab'om. Wa' we alab'om katajin kakipatz kib' are chi keb'e pa waka tem. Are taq xraj we jun ala kateri' b'i chi kij, na xkaj ta le alab'om, xkanaj kanoq utukel.

Xuchaple'j k'u uka'yexik jas taq le k'o pa le alaj juyub'. Chi uwa le xan, xekel apanoq job' raqan atz'yaqib'al, xuquje' job' tzu; are taq wa' uka'yeb'alil, saq, q'an, kaq, rax chi'l q'eq.

Le ala xrilo, chi le alab'om are taq kakikoj le rax katz'yaq, xuquje' kakitor uwi' le jun rax tzu; sib'alaj kaq'equmar uwa le kaj, xa jeri' kub'an jab' ronojel q'ij.

Chaq'ab' are taq xrilo, le alab'om xkitor uwi' le kaq tzu. Xel uloq kaq taq uxaq q'aq' chi upam. Xa jeri' xuchaple'j wululem le kaqulja chi uwa taq le kaj, xuquje' sib'alaj kareperob' taq le xkoyopa'.

Le alab'om xekisik'ij le sutz', la jab' chi'l le xkoyopa' are chi katani' le jab'. K'ate k'u ri' chi kijujunal xe'ok pa taq le jujun kitzu.

Pa jun q'ij chik, le alab'om xkikoj jun q'an atz'yaq chi kijujunal. Are taq xkitor uwi' le jun q'an tzu, xel uloq chi upam jun nimalaj kiqiq'. Pa ri q'ij ri', le alab'om xkiya' jun ulew b'o'j xuquje' jujun ub'aq'wach kinaq che le ala. Xkib'ij chi rech chi kub'an le rikil ruk' taq le kinaq' xkiya'o. Are k'u we ala na xunimaj taj jas le xkib'ij le alab'om, are taq xub'an le rikil xutz'aqatisaj na ruk' nik'aj chik kinaq'. Are taq xpoq'owik, xpulin uloq. Xyaj le jun ala, rumal na xunimaj taj, jas le xkib'ij le alab'om.

Pa le jun q'ij chik, le alab'om xkikoj le saq atz'yaq. Xpakapob' uloq chi uwa taq le kaj, xk'utun uwa le q'ij. Xkib'ij k'u kanoq chi rech le jun ala, chi keb'e na pa jun kiwakatem rech keb' oxib' q'ij, rumal ri' xaq kamatzi'k, na k'o ta kuchap che taq le jas taq k'o kanoq. 

Are k'u we jun ala na kamatzi' taj. Are taq xutukelaj kanoq, xukoj le jun q'eq atz'yaq xuquje' xutor le jun q'eq tzu. Xa jeri' xuchaple'j keb' raqan pa ronojel le uwachulew. Xuquje' xpe jun nimalaj kiqiq', xuchaple'j le jab'. Wuqub' q'ij xub'an wa' we nimalaj k'ax.

Xkiriq k'ax le q'at alab'om chi uq'atexik uwach we k'ax. Xtani' na le jab', k'ate ri' xq'alajin ri nimalaj k'eyowal xya'tajik.

Chqana'taj jujun taq ucholajil we tzijonem rech taq le tzu.

Ruk' jun uwi' qaq'ab' chqak'utunsaj jachike le jab' kaya'tajik are taq kaqakoj le jujun atz'yaq chi'l taq le tzu.

Jas xqeta'maj chi rij le jun tzijomem rech taq le tzu?

Kuk' taq le e wachi'l alab'om, alitomab'; chqatzijoj jas xuk'ulmaj le jun ala katzijox pa le tzijonem.

La k'o jujun taq mul na at nimanaq ta pa kiq'ab' le atat anan, le awajtij? Jachike le jun eta'manik xaweta'maj pa we jun tzijonem?

Ri uq'ijol le saq'ij, chi'l ri uq'ijol le q'alaj chqak'utunsaj pa jun cholb'al q'ij, jachike taq ik' kaya'taj ri saq'ij, jachike taq ik' kaya'taj ri q'alaj.

Pa jun uxaq wuj chqawachib'ej jachike uwach taq chak kaqab'an pa uq'ijol ri saq'ij. Pa jun chik uxaq wuj chqawachib'ej jachike taq chak kaqab'an pa uq'ijol ri q'alaj.

Retz'ab'exik ri uch'etowem le jab' jun chi kech le qachi'l kuk'am ri qab'e, kaq'ax k'ut chi kiwach le e tijoxelab'. We kub'ij: Pa le kaj k'o sutz' .... kajib' uch'etowem. Aninaq kojk'oji' oj kajib' pa jun puq. Jachin k'u le na xuriq ta uk'olib'al pa le puq, kelesax kan pa le etz'anem. Kaqatatab'ej chik jachike rajlab'alil kub'ij le qachi'l pa le jun chik taqanik.

Kaqak'exwachib'ej retz'ab'exik le jab'

Chqab'ana job' puq ri oj tijoxelab'. Le ajtij kujach jujun uka'yeb'al atz'yaq chi'l jujun uka'yeb'al tzu che jun taq puq. Le jujun taq puq kakik'utunsaj uwach le jab' kel uloq pa le jujun tzu, jacha uka'yeb'al le xutzijoj pa le tzijonem.

Chqab'ana' nab'e jun k'utb'al rech le taqanik kab'ixik; k'o jun puq xuriq le q'an tzu. Kojel apanoq chwa le tijob'al, kaqatzukuj jun alaj qaxik'a'y. Ruk' uxer taq wuj kaqab'an jujun alaj taq mochko'y, k'ate k'u ri' kaqaxekeb'a chi utza'm le jun alaj xik'a'y.

Are taq oj k'o chik qonojel, le ajtij kuk'am qab'e chi ub'anik we jun etz'anem. Le ajtij kub'ij jun ka'yeb'al, are k'u le puq kuk'utunsaj le uchak, utz we kuk'ut ruk' alaj taq xojlinem, jujun uwach silab'am che kuq'alajisaj jachike le kak'utunsaxik.
